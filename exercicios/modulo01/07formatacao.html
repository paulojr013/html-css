Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando as tags <b> </b> (não semântica)

Nessa frase temos um termo em destaque usando as tags <strong> </strong> (semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> </i> (não semântico)

Nessa frase temos um termo em ênfase usando a tag <em> </em> (semântica)

Texto Inserido / Sublinhado

Podemos marcar un texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizamos as tags <ins> </ins> (semântico).

Existe também o sublinhado com a tag <u> </u>(não semântico)

Texto Marcado

Podemos também criar textos marcados com a tag <mark> </mark>

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno. Utilizamos as tags <big> </big> (obsoletas) e <small> </small>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Utilizamos as tags <del> </del>.

Texto Sobrescrito

Para inserir coisas do tipo x20+3 utilizamos as tags <sup> </sup>

Texto Subscrito

Para inserir coisas do tipo H2O utilizamos as tags <sub> </sub>
